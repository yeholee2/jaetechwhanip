## 🧭 프로젝트 개요

### 왜 만드는가
- **토스 커뮤니티** — MAU는 있지만 투기성 콘텐츠 난무, SEO 없음, 깊이 없음
- **아하(a-ha.io)** — SEO 강하지만 금융 버티컬 약함, AI 어뷰징으로 신뢰 무너지는 중
- **재테크한입** — 금융 특화 + SEO 되는 + 사람이 실제로 답변하는 신뢰 구조

### 타겟
진지하게 재테크 공부하고 싶은 20-30대 직장인/사회초년생
