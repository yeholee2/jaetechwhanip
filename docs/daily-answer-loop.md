# 하루 1개 답변 운영 루프

## 목적
- 초반 Q&A 서비스의 빈 화면 문제를 줄인다.
- 검색 유입이 생길 만한 질문에 좋은 답변을 쌓는다.
- 아하처럼 질문 상세 페이지가 누적 자산이 되게 만든다.

## 원칙
- 광고 문장보다 실제 도움이 되는 답변을 쓴다.
- 투자 조언은 단정하지 않고 기준과 선택지를 설명한다.
- 답변 끝에는 추가 질문을 유도한다.
- 같은 토픽만 반복하지 않고 주 5개 카테고리를 순환한다.

## 주간 루틴
- 월: 재테크 입문
- 화: 주식·ETF
- 수: 절세
- 목: 보험
- 금: 대출·부채
- 주말: 반응 좋은 질문 보강 또는 쉬기

## 답변 템플릿
1. 질문자의 상황을 한 문장으로 정리
2. 핵심 답변 2~3문장
3. 선택 기준 또는 체크리스트
4. 주의할 점
5. 다음에 물어보면 좋은 질문

## 품질 기준
- 500~900자 권장
- 제목의 핵심 키워드를 자연스럽게 1~2회 포함
- 금융 공포감 조성 금지
- 특정 상품 매수/가입 강요 금지
- 출처가 필요한 제도/세율/정책은 최신 확인 후 작성
